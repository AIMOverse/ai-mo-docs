# AiMo Network - User Guide

## Getting Started

### 1. Create Your Solana Wallet

Before using AiMo Network, you need a Solana wallet:

1. Install the Solana CLI tools: [https://docs.solana.com/cli/install-solana-cli-tools](https://docs.solana.com/cli/install-solana-cli-tools)
2. Generate a new wallet: `solana-keygen new`
3. Fund your wallet with tokens for payments (you'll need Solana tokens for the escrow account)

### 2. Install AiMo CLI

You can install the AiMo CLI tool to generate API keys and manage your account:

```bash
cargo install aimo-cli
```
