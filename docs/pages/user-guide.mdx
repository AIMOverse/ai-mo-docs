# AiMo Network - User Guide

Welcome to AiMo Network, a decentralized AI model routing service that connects you to AI model providers in a peer-to-peer fashion. This guide will help you get started with using AiMo Network's model services.

## Overview

AiMo Network is a decentralized platform that allows you to access AI model services (like text completion, chat, and other AI models) through a unified API. The service operates on a pay-per-use model using blockchain-based payments on the Solana network.

### Key Features

- **Decentralized AI Access**: Connect directly to AI model providers without intermediaries
- **OpenAI-Compatible API**: Use familiar OpenAI API formats for seamless integration
- **Streaming Support**: Real-time streaming responses for chat and completion models
- **Blockchain Payments**: Secure, transparent payments using Solana tokens
- **Multiple Model Providers**: Access various AI models from different providers

## Getting Started

### 1. Create Your Solana Wallet

Before using AiMo Network, you need a Solana wallet:

1. Install the Solana CLI tools: [https://docs.solana.com/cli/install-solana-cli-tools](https://docs.solana.com/cli/install-solana-cli-tools)
2. Generate a new wallet: `solana-keygen new`
3. Fund your wallet with tokens for payments (you'll need Solana tokens for the escrow account)

### 2. Install AiMo CLI

You can install the AiMo CLI tool to generate API keys and manage your account:

```bash
cargo install aimo-cli
```
